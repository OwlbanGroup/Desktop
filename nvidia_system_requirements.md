# NVIDIA Control Panel System Requirements

## Minimum Configuration
- **Operating System**: Microsoft Windows 10 or later
- **Processor**: A 1.5 GHz processor (e.g., Intel Core i3, AMD Ryzen 3)
- **RAM**: 4 GB
- **Disk Space**: 2 GB of free hard disk space

## Recommended Configuration
- **Processor**: A 2.0 GHz processor (e.g., Intel Core i5, AMD Ryzen 5)
- **RAM**: 8 GB
- **Internet Connection**: Required for optimal performance
